Mums in Tech | Week 03

What have we covered la la la so far…

  • What's HTML

    We have now covered HTML, including: what it is; how to write it and why it's important to try and keep the semantic meaning of our HTML whilst separating the visual information and design.

  • The Terminal

    With a harrowing journey through the beggining of the terminal, we have covered how to navigate directories using cd to change directories, ls to list directory contents and have started to feel more confidant when faced with a terminal window.

  • Version control with Git

    The introduction to Git covered why it is used (tracking changes) as well as the impact it can have when working in a team (branches, merging, pushing and pulling). Some have been able to progress to actually publishing their own code onto GitHub which is a fantastic milestone!

  • Styling with CSS

    The way a HTML page looks is powered primarily by the CSS used. There are a huge number of CSS properties and many small differences between web browsers. These are things that are learn over time, the important thing to do is try. Search, checkout examples even inspect other sites to see how they have done it.
